Element(opt_options)
Creates a new Element. All Flora elements extend Element. Element holds the minimum properties need to render the element via draw().
Parameters:
{Object} opt_options Optional
Element options.
{string} opt_options.id Optional, Default: "m-" + Element._idCount
An id. If an id is not provided, one is created.
{Object|function} opt_options.view Optional
HTML representing the Element instance.
{string} opt_options.className Optional, Default: 'agent'
The corresponding DOM element's class name.
{array} opt_options.sensors Optional, Default: []
A list of sensors attached to this object.
{boolean} opt_options.controlCamera Optional, Default: false
If true, camera will follow this object.
{number} opt_options.width Optional, Default: 20
Width
{number} opt_options.height Optional, Default: 20
Height
{number} opt_options.scale Optional, Default: 1
Scale
{number} opt_options.angle Optional, Default: 0
Angle
{number} opt_options.opacity Optional, Default: 0.85
Opacity
{string} opt_options.colorMode Optional, Default: 'rgb'
Color mode. Valid options are 'rgb'. 'hex' and 'hsl' coming soon.
{Array} opt_options.color Optional, Default: null
The object's color expressed as an rbg or hsl value. ex: [255, 100, 0]
{number} opt_options.zIndex Optional, Default: 1
z-index
{number} opt_options.borderWidth Optional, Default: false
borderWidth The object's border width in pixels.
{string} opt_options.borderStyle Optional, Default: false
borderStyle The object's border style. Valid options are 'none', 'hidden', 'dotted', 'dashed', 'solid', 'double', 'groove', 'ridge', 'inset', 'outset', 'inherit'.
{string} opt_options.borderColor Optional, Default: false
borderColor The object's border color expressed as an rbg or hsl value. ex: [255, 100, 0]
{string} opt_options.borderRadius Optional, Default: false
borderRadius The object's border radius as a percentage.
{string} opt_options.boxShadow Optional, Default: false
boxShadow The object's box shadow.
{Object} opt_options.location Optional, Default: The center of the world
The object's initial location.
{Object} opt_options.acceleration Optional, Default: {x: 0, y: 0}
The object's initial acceleration.
{Object} opt_options.velocity Optional, Default: {x: 0, y: 0}
The object's initial velocity.
