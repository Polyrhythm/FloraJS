Class Detail
Oscillator(opt_options)
Creates a new Oscillator. Oscillators simulate wave patterns and move according to amplitude and angular velocity.
Parameters:
{Object} opt_options Optional
Oscillator options.
{Object} opt_options.initialLocation Optional, Default: The center of the world
The object's initial location.
{Object} opt_options.lastLocation Optional, Default: {x: 0, y: 0}
The object's last location. Used to calculate angle if pointToDirection = true.
{number} opt_options.width Optional, Default: 10
Width
{number} opt_options.height Optional, Default: 10
Height
{Object} opt_options.amplitude Optional, Default: {x: 4, y: 0}
Sets amplitude, the distance from the object's initial location (center of the motion) to either extreme.
{Object} opt_options.acceleration Optional, Default: {x: 0, y: 0}
The object's acceleration. Oscillators have a constant acceleration.
{boolean} opt_options.isStatic Optional, Default: false
If true, object will not move.
{boolean} opt_options.isPerlin Optional, Default: true
If set to true, object will use Perlin Noise to calculate its location.
{number} opt_options.perlinSpeed Optional, Default: 0.005
If isPerlin = true, perlinSpeed determines how fast the object location moves through the noise space.
{number} opt_options.perlinTime Optional, Default: 0
Sets the Perlin Noise time.
{number} opt_options.perlinAccelLow Optional, Default: -0.075
The lower bound of acceleration when isPerlin = true.
{number} opt_options.perlinAccelHigh Optional, Default: 0.075
The upper bound of acceleration when isPerlin = true.
{number} opt_options.offsetX Optional, Default: Math.random() * 10000
The x offset in the Perlin Noise space.
{number} opt_options.offsetY Optional, Default: Math.random() * 10000
The y offset in the Perlin Noise space.
