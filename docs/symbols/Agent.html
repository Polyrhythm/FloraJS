Agent(opt_options)
Creates a new Agent. Agents are autonomous objects affected by World forces (gravity, wind) and Proximity forces (Attractors, Repellers, Liquid). Because their primary purpose is to navigate their World, they carry navigational properties like 'avoidEdges', 'maxSteeringForce' or 'turningRadius' which can be manipulated to adjust their observed behavior. They can also carry Sensors which react to Stimuli and return information about the World.
