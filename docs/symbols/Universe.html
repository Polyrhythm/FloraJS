Universe(opt_options)
Creates a new Universe.
Parameters:
{Object} opt_options Optional
Options.
{boolean} opt_options.isPlaying Optional, Default: true
Set to false to suspend the render loop.
{boolean} opt_options.zSorted Optional, Default: false
Set to true to sort all elements by their zIndex before rendering.
{boolean} opt_options.showStats Optional, Default: false
Set to true to render stats on startup.
{boolean} opt_options.isDeviceMotion Optional, Default: false
Set to true add the devicemotion event listener. Typically use with accelerometer equipped devices.
